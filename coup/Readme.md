# Coup
https://github.com/fluffyOkitsune/Coup

# 概要
みんなで騙し合うきつねっぽいカードゲームです。

# ルール
* プレイヤーはそれぞれシャッフルされたカードを2枚ずつ持ち、ゲームを開始する。この際、余ったカードは山札とする。
* 手番のプレイヤーはカードによって異なる『行動』を行うことができる。
* 特定の『行動』は特定のカードによって『妨害』されうる。『妨害』された場合、カードは効果を発揮しない。
* 『行動』ないし『妨害』を行う際、所持するカードは偽ることができる。偽っている惧れがある場合は『ダウト』と宣告することで、その旨を指摘することができる。
* 『ダウト』を宣告されたプレイヤーは、直ちにカードを公開し、正当なカードを所持していることをプレイヤー全員に示さねばならない。
* 『ダウト』が正当である（カードを偽って『行動』ないし『妨害』を行っていた、およびカードの公開を拒否した）場合は、宣言されたプレイヤーはカードを1つ失う。
* 『ダウト』が誤りであった場合は、『ダウト』を宣告したプレイヤーがカードを1つ失う。
* 『行動』、あれば他プレイヤーの『ダウト』宣告、あれば他プレイヤーの『妨害』の手順を経たら次のプレイヤーへと手番が移る。
* 金貨を10枚以上持つプレイヤーは自分の手番で行動『クー』をしなければならない。
* カードを2枚とも失ったプレイヤーは敗北し、手番を得る権利を失う。
* 最後までカードを所持していたプレイヤーが勝者である。

# カード
* 公爵	：徴税可能／援助を妨害可能
* 刺客	：暗殺可能
* 大使	：交換可能／強奪を妨害可能
* 船長	：強奪可能／強奪を妨害可能
* 女伯	：暗殺を妨害可能

#行動
金貨が必要な行動は、それが不足している場合、その行動を行うことはできない
* 収入	：金貨を1枚獲得／妨害不可
* 援助	：金貨を2枚獲得
* 徴税	：金貨を3枚獲得／妨害不可
* 暗殺	：金貨を3枚失い、対象のプレイヤーのカードを1枚無効可
* 交換	：山札からカードを2枚引き、自分だけ中身を確認の後、そのうち一方を手札と交換
* 強奪	：対象のプレイヤーから金貨を2枚奪う
* クー	：金貨を7枚失い、対象のプレイヤーのカードを1枚失わせる
